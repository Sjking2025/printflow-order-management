### Task 2-4: Entity fields
- `Shop.copyModifyWindowMins` — configurable per shop (1-30 min, default 5)
- `Order.copyModifyExpiresAt` — calculated at creation
- `OrderDocument.copiesModifiedAt` — null = not yet modified

### Task 5: Order creation
- `OrderService.createOrder()` sets `copyModifyExpiresAt = now + copyModifyWindowMins`

### Task 6: Validation rules in OrderDocumentService.updateCopies()
1. copies >= 1
2. document exists + belongs to customer
3. order must be `PENDING`
4. current time < `copyModifyExpiresAt` (window check)
5. `copiesModifiedAt` must be null (once per document)
6. `newCopies > current copies` (increase-only)

### Task 7: Typed exception handling
- New `CopyModificationException` with `ErrorCode` enum (`INCREASE_ONLY`, `WINDOW_EXPIRED`, `ALREADY_MODIFIED`, `NOT_PENDING`)
- Controller catches the typed exception and returns 409 with error code + message
